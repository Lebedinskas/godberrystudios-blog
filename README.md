# Godberry Studios Blog

Built with [Hugo](https://gohugo.io) and deployed on [Cloudflare Pages](https://pages.cloudflare.com).

## Local Development

```bash
hugo server -D
```

## Adding a New Post

Create a new file in `content/posts/`:

```bash
hugo new posts/my-new-post.md
```

## Deployment

Automatic via Cloudflare Pages — push to `main` and it deploys.
