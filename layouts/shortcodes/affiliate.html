{{/* Affiliate link shortcode — Godberry Studios Two usage modes: 1. Cloaked (preferred for vendor links — bypasses ad-blocker filters): {{< affiliate slug="make" label="Make" >}} Requires a matching line in static/_redirects: /go/make https://www.make.com/en/register?pc=godberry 302 2. Direct (legacy — vendor without a slug entry): {{< affiliate url="https://partner.example.com/?ref=godberry" label="Vendor" >}} Some browser extensions hide direct affiliate URLs. Prefer the cloaked variant for any link that needs to convert. Notes: - rel="sponsored nofollow" is required by Google for paid links. - The small "(affiliate)" marker is visible by design — readers must see it. - target="_blank" + rel="noopener" so vendor pages open cleanly. - Class names use neutral "vendor-ref" / "post-disclosure-note" rather than "affiliate-*" so cosmetic ad-block filter lists don't hide them. The rel="sponsored nofollow" attribute and visible "(affiliate)" marker still satisfy Google + FTC requirements. */}} {{- $slug := .Get "slug" -}} {{- $url := .Get "url" -}} {{- $label := .Get "label" -}} {{- $href := "" -}} {{- if $slug -}} {{- $href = printf "/go/%s" $slug -}} {{- else if $url -}} {{- $href = $url -}} {{- end -}} {{- if and $href $label -}} {{ $label }} (affiliate) {{- else -}} [affiliate shortcode missing slug/url or label] {{- end -}}
